Maintained/Welcome

Welcome to the Maintained dashboard, your management tool for creating and updating badges for projects. Make dynamic badges that pull data from sources like GitHub, Travis CI, and custom API endpoints, share projects with teams, and more. Let's take a tour of some basics to get started.

Badges are highly customizable, dynamically updatable elements. Below, you'll see we've created a starter badge and project for you. Badges let you display any text you wish in fields, in a variety of styles. The default number of fields is two, and the minimum is one, but you can add up to 25 fields for some seriously crazy badging!

Badge customization can be done by clicking the gear icon to the right of the badge. You can change the colors, add icons, and add or remove badge fields. Advanced settings include Dynamic Value Sources and enabling dynamic redirects with Link Direct. Learn more about advanced customization features.

Maintained organizes your badges into projects. You can create new projects by clicking "Create New" in the top left corner. Navigate your projects on the sidebar to the left. Default settings, like badge style and color, can be configured by clicking the gear icon on a project. Projects can also be shared with other Maintained users.
